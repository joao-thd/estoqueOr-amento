# 📦 Projeto CRUD com C#, ORM e MySQL

Este é um projeto de exemplo de uma API CRUD (Create, Read, Update, Delete) desenvolvida com C# (.NET Core), utilizando um ORM (Entity Framework Core) e banco de dados MySQL.

## 🚀 Funcionalidades

- ✅ Criação de registros  
- 🔍 Leitura (todos ou por ID)  
- ✏️ Atualização de dados  
- 🗑️ Exclusão de registros  
- 🔗 Conexão com banco de dados MySQL via ORM  

## 🛠️ Tecnologias Utilizadas

- C# (.NET Core)  
- Entity Framework Core  
- MySQL  
- Swagger (para documentação da API)  
- RESTful API  
- Visual Studio
